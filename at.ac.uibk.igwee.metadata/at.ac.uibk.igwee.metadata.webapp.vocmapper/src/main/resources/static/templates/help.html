How to use this application

Main idea

Creating an edition is hard enough, but if one wants to link up people, places and corporates with controlled vocabularies, the task becomes tedious. This application should help people link up the internal vocabularies (identified by a key- or a name-value) with controlled vocabularies from other authorities, e.g. Geonames, Virtual Internet Authority File, or Wikidata. (More authorities may be available soon.)

For this purpose, one needs a link table from the internal key value to the URI provided by these authorities. This table may also be formulated using RDF with the predicate owl:sameAs.

Here are the working steps:

  • Make a table with your favorite spread sheet application
  • Upload the table to the application.
  • Use the Prequery tab to prequery the vocabularies. The application will work the table line by line and queries the authorities for controlled vocabularies.
  • At anytime, one can use the Download button to download the progress, the progress will be saved as an XML file. Right now, this is the only way of exporting the data.
  • Use the Edit tab to edit the results. Specifically one should fix a certain result of the query. By doing so, a link is established between the internal key value with an external controlled vocabulary. If is also possible to requery a query, to reformulate the query string, or to add a controlled vocabulary URI.
  • Export the data as XML and use the data for your own projects.

Spread Sheet Table

Please insert the query data into an Excel table, using this template.

The Name columns contain the key value (or the name value) of your projects, e.g. "Einstein, Albert".

The Query columns contain the query string, e.g. "Albert Einstein". Please note that the query string will be sent to the different services. So, it may be required to discard accents.

The Authorities columns contain the desired authorities that are queried with this application. Valid values are : "viaf", "geonames" and "wikidata". If this field is left blank, then all services available will be used. If you want to specify more than one service, use " " (space) to delimit the different authorities.

The Type columns specify the types of the vocabularies. Valid values are "Person", "Place" and "Corporate". If you are not sure, you can leave the field blank.

The Info columns contain just additional information that might be used to specify the vocabularies. E.g. one can fill in "Physicist" to announce that the person in query is a physicist; or one can add birth- and deathdates and other valuable information. This column will not be used for query, but it will be displayed in the editing window.
