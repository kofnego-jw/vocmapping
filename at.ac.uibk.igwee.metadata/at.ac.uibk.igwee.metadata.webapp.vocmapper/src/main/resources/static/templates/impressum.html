Used Libraries

The following libraries are used to create this application.

  • For the backend
    • Spring Framework
    • Apache Http Components
    • Saxon XSLT Library
    • XStream Serializer
    • Apache Commons (IO, Upload, Logging)
    • slf4j
    • Jackson JSOn Serializer
  • Authorities used
    • Geonames
    • Virtual Internet Authority File (VIAF)
    • Wikidata
  • For Frontend
    • Angular JS
    • Angular JS UI Router
    • JQuery
